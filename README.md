# Hush! I remember
The app is for those who constantly doubt whether they have turned off the iron, closed the door, or performed another important action.
Приложение предназначено для тех, кто постоянно сомневается, выключил ли он утюг, закрыл ли дверь или выполнил другое важное действие.
